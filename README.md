# Tic-Tac-Toe Game (C++)

This is a simple console-based Tic-Tac-Toe game developed using C++.
The game allows two players to play against each other in turns.

## Features
- 3x3 game board
- Two players: X and O
- Turn-based gameplay
- Win and draw detection
- Option to play again

## How to Run
1. Copy the code into a C++ compiler
2. Compile the program
3. Run the executable file
4. Follow the on-screen instructions

---------
